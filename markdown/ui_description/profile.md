# Profile tab

Visible only to logged-in users.

## Account panel

- **Username** — editable field pre-filled with the current username. Save button updates it server-side; must remain unique.
- **Password** — two fields (current password, new password). Requires the correct current password before accepting a change.

## Dota 2 Player panel

- Optional link between the user's account and an OpenDota account ID (the player's real Kanaliiga account).
- When a valid ID is saved and the player exists in the database, the player's name and avatar are shown as a preview.
- Intended for future features (e.g. auto-granting a card representing the user's own player).
