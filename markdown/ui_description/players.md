# Players tab

Visible to everyone.

## Players panel

Table of all players who have participated in any ingested league. Columns: Player (avatar + name, clickable → Player detail modal), Team, Matches, Avg pts, Total pts.

- **Filter** — text field filters rows in real time by player name or team name.

## Player detail modal

Opened by clicking any player name (in this tab or elsewhere). Shows:

- Player avatar, name, and current team.
- Summary stats (total matches, average points, total points).
- **Match history table** — one row per match, columns: Date, Fantasy pts, K/A/D, GPM, Wards, Tower dmg.
